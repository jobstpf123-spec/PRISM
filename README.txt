WHAT THIS IS
------------------------------------------------------------------------
PRISM is the employee behavioural assessment. It runs entirely on your
computer in a normal web browser. There is nothing to install, no
internet connection required, and no technical setup.

Version 2.0 measures 45 behavioural constructs (Leadership Style,
Conflict Resolution, Workplace Personality) using a bank of 225 items
across five question types. Every employee's assessment is assembled
automatically from that bank based on their grade and department, so
no two employees receive an identical set of questions, while every
assessment remains equivalent in what it measures.


WHAT IS IN THIS FOLDER  (keep all four files together)
------------------------------------------------------------------------
  PRISM.html ............ the application — this is the file you open
  xlsx.mini.min.js ...... required helper file (creates the Excel export)
  Tri-Pack_Logo.png ..... the Tripack logo shown in the app
  README.txt ............ this file

Please keep these files together in the same folder. If you move
PRISM.html on its own, the logo and the Excel export may stop working.


HOW TO START
------------------------------------------------------------------------
  1. Double-click  PRISM.html
  2. It opens in your web browser (Google Chrome or Microsoft Edge
     work best; Firefox is also fine).

That is all. No login to any website, no internet needed.


FOR EMPLOYEES  (taking the assessment)
------------------------------------------------------------------------
  1. Enter your Employee ID and your name, exactly as HR has them.
  2. Read the short instructions.
  3. Work through the three modules (Leadership, Conflict, Personality).
     Choose the answer closest to what you would really do. There are no
     right or wrong answers and no time limit.
  4. When you finish you will see a confirmation message. You are done —
     you may close the window. Employees never see any scores or results.


FOR HR ADMINISTRATORS  (collecting the results)
------------------------------------------------------------------------
  1. On the sign-in screen, click "Administrator access" at the bottom.
  2. Enter the administrator passcode:   TRIPACK-HR
     (Ask IT to change this before rolling out, if you wish.)
  3. In the Administrator Portal you can:
       - Load your Employee Master list (a CSV file) so the app can
         check who is signing in.
       - See how many people have completed the assessment.
       - Click "Export results (Excel)" to download the file
         TTIP_Assessment_Results.xlsx
  4. Give that Excel file to whoever runs the TIE workbook (TIE.xlsx).
     Paste its rows into the "Psychometric Scores" sheet. The TIE then
     produces the T-scores, the Behavioural Signature and the report.


GOOD TO KNOW
------------------------------------------------------------------------
  - Results are stored only in the browser on that computer until you
    export them. Export regularly and keep the Excel file safe.
  - The assessment can be run on one shared computer for many employees:
    after someone finishes, reload PRISM.html for the next person.
  - No data is ever sent over the internet.


IF SOMETHING DOES NOT WORK
------------------------------------------------------------------------
  - Logo not showing, or export button does nothing:
      make sure all four files are still together in the same folder,
      and that none have been renamed.
  - Open with Google Chrome or Microsoft Edge if another browser behaves
    oddly.
  - Still stuck: contact your IT team and show them this README.
